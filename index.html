API 설정

Google Apps Script 웹앱 URL을 입력하면 모든 데이터가 구글 시트와 연동됩니다.

웹앱 URL 설정

Apps Script 배포 후 생성된 웹앱 URL을 붙여넣으세요. 저장하면 브라우저에 기억됩니다.

현재 저장된 URL 없음

기본 내장 URL (변경 없이 바로 사용 가능)
AKfycbxFkZKlzMY7GvBVDiMuKjnwEkaPDZzENNnqaKdCr_o86… (클릭하여 입력란에 적용)

배포 가이드

  1. 제공된 Code.gs 파일을 구글 시트 → 확장 프로그램 → Apps Script에 붙여넣기
  2. 상단 메뉴 배포 → 새 배포 클릭
  3. 유형: 웹 앱 선택
  4. 실행 계정: 나, 액세스 권한: 모든 사용자(익명 포함) → 배포
  5. 생성된 웹앱 URL을 위 입력창에 붙여넣고 저장
  6. 스프레드시트에서 🛠 마케팅 대시보드 → 시트 초기화 실행 (최초 1회)
  7. 업로드 파일은 마케팅 인프라 구글 드라이브 폴더에 자동으로 저장됩니다.
